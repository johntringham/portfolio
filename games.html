Games

I have been making games and experimental visual programs for as long as I can remember. I learnt how to program back when I was 13-14, using Adobe Flash (then Macromedia Flash), in order to make games and visual experiments.

I'm a full-indie solo game developer, after working as a Lead UI Developer for a software firm for the last 6 years. I largely use Unity, but have also worked with a large number of other frameworks.

Roamer

Roamer is a game I have been working on solo during the Coronavirus pandemic. It is a moody and atmospheric 3D stealth/puzzle/adventure game.

I have been working on all areas, including all programming, designing, 3D modelling, texturing, rigging, procedural animation, enemy AI, effects, audio generation, and writing. It's being developed in Unity, using URP, Shadergraph and Blender.

Two Player

Two Player is a short single player video game about the social complexities of playing couch co-op video games. It includes multiple endings and branching dialog, which is controlled based on the choices you make in a in-game FPS, and shows how these choices affect the real life social situation you are in.

It was named in Game Curators' Game Of The Year 2020: Game Jam Edition list, and was also featured in Alpha Beta Gamer and Warp Door.

Two Player was created in 4 days as part of the 7DFPS Game Jam. It was made entirely by myself, using Unity and Doodle Studio 95.

You can play it for free in-browser on itch.io

Evaze

Evaze is an abstract 2D Arcade-Puzzle game, made in Unity.

In Evaze, the player navigates a malleable labrynth - at any point a tile can be slid in and push all the other tiles accross, in order to create new routes to orbs and to block off enemies.

Evaze is a casual puzzle-action mobile game. It can be downloaded for both Android and iOS.
